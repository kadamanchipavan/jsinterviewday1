7.Explain call(), apply() and, bind() methods. Give an example of call(), apply(), bind()

  • Call is a method in function that helps us to replace the value of 'this' inside a function with whatever value we want. Some additional arguments can alse be given for the variables inside the function.
  • Apply is very similar to the call method. The only difference is that in apply we can pass an array as an argument list.
  • Bind is a method in function that helps us to create another function that we can execute later as per our requirement.
